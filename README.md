# sentiment-portfolio-moex

Код к магистерской диссертации «Формирование портфеля акций на российском фондовом рынке с применением сентимент-анализа новостного фона».

## Конвейер

```
Сбор данных (котировки + новости)
      ↓
Эмбеддинги новостей + дедупликация
      ↓
Поиск тикеров в новостях
      ↓
   ┌─ Сентимент по тикерам (GeRaCl) ──────────────┐
   │                                              ├─→ Портфельные модели
   └─ Макро/гео: классификация + сигналы +         │   (LightGBM, walk-forward)
      сентимент (YandexGPT / GeRaCl) ─────────────┘
        (→ фичи market_sent / sector_sent)

Проверка гипотез (event study) — отдельно, на сентименте по тикерам
```

## Структура репозитория

### Сбор данных
- **`Парсер рыночных данных/`** — котировки MOEX (мин. свечи, IMOEX), Brent, USD/RUB, ставка ЦБ, дивиденды, капитализация
- **`Парсер тг каналов/`** — парсер 29 Telegram-каналов через Telethon → SQLite
- **`Парсеры лент/`** — скрейперы новостных лент (Коммерсантъ, Лента, РИА) на aiohttp/BeautifulSoup → SQLite

### Обработка новостей
- **`Эмбеддинги и дедупликация/`** — векторные эмбеддинги новостей (Sber Giga-Embeddings) и дедупликация дублей между каналами
- **`Поиск тикеров в новостях/`** — поиск упоминаний компаний (по `tickers.json`) с учётом морфологии: отдельно для Telegram и новостных лент
- **`Сентимент/`** — сентимент новостей (GeRaCl: neg/neu/pos) для Telegram и СМИ
- **`Макро и гео новости/`** — классификация макро/геополитических новостей и извлечение сигналов (scope рынок/сектор, затронутые тикеры) через YandexGPT; сентимент этих новостей — через GeRaCl

### Анализ и модель
- **`Анализ гипотез/`** — event-study проверка гипотез: H1 (Telegram опережает СМИ), H2 (негатив сильнее позитива), H3 (чувствительность крупных компаний), H4 (секторный переток). С отчётами.
- **`Портфельные модели/`** — итоговые портфели **v1** (простое сентимент-правило), **M1** (базовая ML), **M2** (+ softmax-взвешивание), **M3** (расширенная: Brent, дивиденды, выход в кэш). Конвейер сборки фич + отчёты бэктестов. Подробности — в README внутри папки.

### Справочники
- **`tickers.json`** — тикер → название → сектор (50 компаний MOEX)


## Стек

Python · Telethon · aiohttp + BeautifulSoup · LightGBM · polars / pandas

Модели:
- Эмбеддинги — `ai-sage/Giga-Embeddings-instruct`
- Сентимент — `deepvk/GeRaCl-USER2-base`
- Классификация макро/гео — `yandex/YandexGPT-5-Lite-8B-instruct` (GGUF, Q4_K_M)
